# EvoAgentX

Self-Evolving AI Agent Framework with Medical AI Integration

## Quick Start

```bash
pip install evoagentx
```

## Features

- Medical AI Tools (PubMed, ClinicalTrials, DrugBank)
- EvoX Evolution Bridge
- Workflow Engine
- RAG System
- HITL (Human-in-the-Loop)
- Benchmark Suite
