⬡ Code Playground

Build medical agents in minutes

EvoAgentX provides a clean, Pythonic API for creating intelligent agents with access to medical databases and evolutionary optimization.

  • Import tools directly from evoagentx.tools
  • Create agents with LLMBackbone integration
  • Compose multi-agent workflows with YAML
  • Evolve strategies with EvoX bridge
  • Built-in RAG for knowledge grounding
from evoagentx.tools import PubMedTool, DrugBankTool from evoagentx.agents import Agent from evoagentx.models import LLMBackbone # Initialize the LLM backbone llm = LLMBackbone(model="gpt-4o", temperature=0.1) # Create medical research tools pubmed = PubMedTool(max_results=10) drugbank = DrugBankTool(api_key="your-key") # Create a medical research agent agent = Agent( name="medical_researcher", llm=llm, tools=[pubmed, drugbank], system_prompt="""You are a medical research assistant. Search literature and drug databases to answer clinical questions.""" ) # Run a research query result = agent.run( "What are the latest treatments for treatment-resistant depression?" ) print(result.output)
from evoagentx.workflow import Workflow from evoagentx.agents import Agent # Define a multi-agent workflow workflow = Workflow(name="clinical_review") # Add agents as workflow steps workflow.add_step( agent="literature_searcher", input="{query}" ) workflow.add_step( agent="evidence_analyzer", input="{previous.output}" ) workflow.add_step( agent="report_writer", input="{previous.output}", condition="confidence > 0.8" ) # Execute the workflow result = workflow.run( query="efficacy of ketamine for PTSD" )
from evoagentx.evolution import EvoXBridge from evox.algorithms import PSO, DE # Connect to EvoX evolutionary framework bridge = EvoXBridge( algorithm=DE(pop_size=50, dimension=20), fitness_fn="task_accuracy" ) # Evolve agent prompt strategies best_strategy = bridge.evolve( target="agent.prompt_template", generations=100, eval_dataset="medical_qa_v2" ) print(f"Best fitness: {best_strategy.fitness}") print(f"Generations: {best_strategy.generations}")
⬡ Quick Start
Up and running in 60 seconds
Install EvoAgentX and create your first evolving medical agent.

Step 1 — Install

pip install evoagentx

Step 2 — Create Your Agent

from evoagentx.tools import PubMedTool from evoagentx.agents import Agent from evoagentx.models import LLMBackbone agent = Agent( name="researcher", llm=LLMBackbone(model="gpt-4o"), tools=[PubMedTool()] ) print(agent.run("Latest papers on CRISPR?"))

Step 3 — Run

python my_agent.py
